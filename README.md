# AgroBalance

## Produzindo mais, desperdiçando menos

O **AgroBalance** é um projeto desenvolvido para o **Concurso Agrinho 2026**, com o tema:

> **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente."**

O objetivo do site é conscientizar sobre o desperdício de água na agricultura e apresentar soluções tecnológicas capazes de aumentar a eficiência no campo, reduzindo impactos ambientais sem comprometer a produtividade.



##  Objetivos do Projeto

* Informar sobre o consumo de água na agricultura.
* Mostrar os impactos ambientais causados pelo desperdício.
* Apresentar soluções sustentáveis para o uso racional da água.
* Demonstrar como a tecnologia pode contribuir para a preservação dos recursos naturais.
* Incentivar práticas agrícolas mais sustentáveis.



##  Funcionalidades

### Calculadora de Consumo de Água

Permite estimar o consumo anual de água de uma plantação com base na área cultivada e no tipo de cultivo.

A calculadora também apresenta uma estimativa de economia de água utilizando sistemas de irrigação sustentável.

### Quiz Ambiental

Quiz interativo com perguntas sobre:

* Sustentabilidade
* Agricultura
* Uso consciente da água

Ao final, o usuário recebe sua pontuação e uma mensagem de desempenho.

### Recursos de Acessibilidade

O site possui:

* Aumento de fonte
* Redução de fonte
* Modo de alto contraste



##  Tecnologias Sustentáveis Apresentadas

* Irrigação por gotejamento
* Captação da água da chuva
* Sensores inteligentes
* Monitoramento digital
* Agricultura de precisão
* Drones agrícolas
* Sensores de umidade
* Inteligência artificial
* Energia solar rural



##  Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

Nenhuma biblioteca externa foi utilizada, tornando o projeto totalmente desenvolvido com tecnologias web fundamentais.



##  Responsividade

O AgroBalance foi desenvolvido para funcionar em:

* Computadores
* Tablets
* Smartphones



##  Mensagem Final

A produção agrícola é essencial para a sociedade, mas também exige responsabilidade ambiental. Com planejamento, inovação e uso consciente dos recursos naturais, é possível produzir mais, desperdiçar menos e construir um futuro sustentável para todos.
